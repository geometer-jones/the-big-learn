# Curriculum

This file defines the initial text canon for The Big Learn.

The goal is not to cover all of Chinese literature. The goal is to choose a small set of texts that match the project's root-first teaching thesis.

## Curriculum Principles

- Start with texts that shaped the written tradition.
- Prefer short, dense, teachable works over giant reading loads.
- Keep the first canon narrow enough to encode well in the six-layer record.
- Separate foundational texts from later literary expansion.
- Be honest that this is a path into the written tradition first, not modern spoken Mandarin first.

## The Five Tracks

The initial curriculum set is:

1. Four Books
2. `Sunzi Bingfa`
3. `Daodejing`
4. `San Zi Jing` / `Qian Zi Wen`
5. `Sanguo Yanyi`

These are not equal in priority. They serve different jobs.

## 1. Four Books

Works:

- `Da Xue`
- `Zhong Yong`
- `Lunyu`
- `Mengzi`

Role:

- main curriculum spine
- root-first entry into classical Chinese thought
- primary source for the first reading workflows

Why it belongs:

- foundational in the educated tradition
- compact enough to study closely
- rich in recurring concepts and patterns
- aligned with the project's thesis better than phrasebook-style content

Start here:

- `Da Xue`

Then:

- `Zhong Yong`
- `Lunyu`
- `Mengzi`

## 2. Sunzi Bingfa

Role:

- early bonus track
- compact strategic prose
- good second-stage text after the learner has tasted the Four Books

Why it belongs:

- short, structured, memorable
- works well line by line
- exposes the learner to concise classical phrasing
- culturally durable and widely referenced

Teaching note:

- use it as a high-density reading and explanation text, not as a separate "business wisdom" gimmick

## 3. Daodejing

Role:

- early bonus track
- philosophical compression track

Why it belongs:

- foundational text with huge downstream influence
- excellent fit for the six-layer line model
- forces careful reading and multiple levels of interpretation

Teaching note:

- this should not be the main beginner spine
- it is too compressed and ambiguous for many learners to use as the first rail

## 4. San Zi Jing / Qian Zi Wen

Role:

- pedagogical bridge track
- lighter on-ramp into literacy patterns and memorization

Why it belongs:

- historically used for foundational learning
- easier to use for repetition and recall drills
- useful when a learner needs more structure before or alongside the classical canon

Project decision:

- treat this as one curriculum tier for now
- final ordering between `San Zi Jing` and `Qian Zi Wen` can be decided later

Teaching note:

- this track is especially useful for flashcard generation and short idle-time sessions

## 5. Sanguo Yanyi

Role:

- later expansion track
- culture, idiom, and narrative literacy pack

Why it belongs:

- massive cultural footprint
- rich source of references, names, scenes, and phrase patterns
- good bridge from compact classical study to broader literary competence

Teaching note:

- do not treat this as a foundational starter text
- it belongs after the learner has a framework

## Recommended Sequence

The default project sequence should be:

1. `Da Xue`
2. the rest of the Four Books
3. `Sunzi Bingfa`
4. `Daodejing`
5. `San Zi Jing` / `Qian Zi Wen` as support or alternate ramp
6. `Sanguo Yanyi`

The ordering is about pedagogy, not prestige.

## How This Connects To The Product

Each curriculum line should be encoded in six stored layers:

1. Traditional Chinese
2. Simplified Chinese
3. Zhuyin
4. Pinyin
5. English line-level literal gloss
6. English holistic translation

Learner-facing guided-reading views should render each annotated line in this fixed sequence:

1. Full-line simplified Chinese
2. Full-line English translation
3. A six-column table:
   `Index`, numbering each character row from 1 through the full line length
   `Chinese`, with simplified primary and traditional in parens when it differs
   `Reading`, with pinyin primary and zhuyin in parens
   `English Definition`, using semicolon-separated context-free character definitions where needed
   `Chinese Phrase`, spanning multiple character rows when the stored segment covers more than one character
   `English Phrase Translation`, aligned to the same phrase spans
4. The same full-line English translation again
5. The same full-line simplified Chinese again

That lets the same source material power:

- guided reading
- line-grounded questions
- direct answers
- flashcard generation
- review drills

## What This Is Not

- not a claim that these texts are the only way to learn conversational Mandarin first
- not a complete Chinese canon
- not a substitute for modern listening and speaking work

This is a deliberate root-first reading curriculum for the kind of learner this project is serving.
